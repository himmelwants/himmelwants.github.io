# 2. JOIN 문법


    #일반 조인 
    SELECT <속성들>                    
    FROM 테이블1, 테이블                            
    WHERE <조인조건> AND <검색조건>
    
    SELECT <속성들>                    
    FROM 테이블1
    INNER JOIN 테이블2 ON <조인조건>
    WHERE <검색조건>
    
SQL 문에서는 주로 동등 조인을 사용한다. 일반 조인문에서는 둘 중 하나를 사용한다. 


    #외부조인|
    SELECT <속성들> 
    FROM 테이블1 
    {LEFT / RIGHT /FULL [OUTER]} JOIN 테이블2 ON <조인조건> 
    WHERE <검색조건> 

외부조인은 FROM 절에 조인 종류를 적고 ON을 이용하여 조인조건을 명시함.
